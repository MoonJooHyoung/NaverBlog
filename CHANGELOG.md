# 변경 이력

## [1.1.0] - 2024

### 추가된 기능
- **랜덤 딜레이 기능**: 자연스러운 동작을 위한 랜덤 딜레이 적용
- **재시도 로직**: 포스팅 실패 시 자동 재시도 (최대 3회)
- **포스팅 히스토리 관리**: 모든 포스팅 기록 저장 및 통계 제공
- **설정 검증**: 시작 전 설정 파일 유효성 검사
- **URL 추출**: 포스팅 성공 후 자동으로 URL 추출 및 저장
- **네이버 블로그 특화 기능**:
  - 공개/비공개 설정
  - 댓글 허용/비허용 설정
  - 예약 발행 시간 설정
  - 썸네일 이미지 설정

### 개선된 기능
- 로그인 성공 확인 로직 강화
- Selenium 동적 대기 개선
- 타입 힌트 추가
- 하드코딩 값 상수화
- 에러 처리 강화
- 환경변수 지원 (보안 강화)

### 수정된 버그
- 빈 except 문 수정
- 로그인 확인 로직 개선
- 이미지 업로드 로직 보완

## [1.0.0] - 초기 버전
- 기본 포스팅 기능
- AI 원고 생성
- SEO 최적화
- 이미지 처리
- 스케줄러 기능
